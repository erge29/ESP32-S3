# ESP32-S3
Rancangan PCB Robot KSRI
Repository ini berisi rancangan PCB dan skematik (SCH) untuk sistem robot berbasis ESP32-S3 yang dirancang sebagai kontrol utama sensor dan aktuator. Sistem ini mengintegrasikan Multiplexer I2C TCA9548A untuk manajemen banyak perangkat I2C, enam sensor ToF VL53L0X untuk deteksi jarak, satu modul kompas CMPS14 untuk orientasi, serta dua TF Lidar untuk pengukuran jarak jarak menengah. Pada sisi aktuator, sistem mengendalikan satu servo EMAX ES3054 dan sembilan belas servo Dynamixel AX-12 sebagai penggerak utama mekanisme robot. Catu daya menggunakan baterai Turnigy 5000mAh 3S 20C LiPo Pack yang diturunkan tegangannya melalui buck converter XL4016 untuk kebutuhan servo dan sistem daya utama, serta regulator AMS1117 untuk suplai 3.3V ke rangkaian logika. Desain PCB difokuskan pada distribusi daya arus tinggi yang stabil, pemisahan jalur logika dan power, serta optimasi layout komunikasi I2C dan UART agar sistem bekerja andal dalam kondisi beban dinamis tinggi.
